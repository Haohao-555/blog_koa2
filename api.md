# 项目接口

## 一、用户登录 
### 1、请求方式 POST
### 2、请求URL /api/user/login
### 3、请求参数
| 字段名 | 注释 | 
| :----:| :----: | 
| username | 用户名 | 
| password | 密码 | 

--- 

## 二、获取博客列表 （需要进行登录验证）
### 1、请求方式 GET
### 2、请求URL /api/blog/list
### 3、请求参数 （以下参数可传可不传）
| 字段名 | 注释 | 
| :----:| :----: | 
| author | 作者 | 
| keyword | 关键字 |

---

## 三、获取博客详情
### 1、请求方式 GET
### 2、请求URL /api/blog/detail
### 3、请求参数 
| 字段名 | 注释 | 
| :----:| :----: | 
| id | 博客的ID （添加到url后面）| 

---
